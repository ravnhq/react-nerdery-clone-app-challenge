## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone <repo_url>
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Create a new `.env` file in the root of the project and enter:
   ```js
   const API_URL = 'API_URL'
   const API_KEY = 'API_KEY'
   ```
